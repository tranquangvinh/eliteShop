Chatfuel Customer Chat plugin lets you install a free Facebook chat directly on your WordPress site page.

== Description ==
Chatfuel Customer Chat plugin lets you install a free Facebook chat directly on your WordPress site page. With one click, your website visitors can have a Facebook Messenger conversation with you. A native Facebook chat button is easily recognizable by anyone and encourages trustful communication with all the features that a Chatfuel Customer Chat can offer.

What makes Chatfuel Customer Chat plugin stand out?
* Easy setup for any Facebook user, no programming skills needed.
* Direct communication with users directly on the page, without leaving the site.
* Instant receiving and replying to messages so your customers aren’t lost.
* Possibility to enhance the chat with a customizable chatbot to automate responses to customer questions and reduce the amount of support agents.

[youtube https://www.youtube.com/watch?v=TKjCB5lWAV4]

== Installation ==
1. Upload the plugin to your FTP or other directory, then install it through the Plugins menu in your WordPress Dashboard.
2. To use the Chatfuel Customer Chat, you need to have a Facebook Page.
3. To enhance your users’ chat experience with a chatbot, contact  [www.masterofcode.com](http://www.masterofcode.com/) for implementing a chatbot for you, or go to [www.chatfuel.com](http://www.chatfuel.com/) to create a chatbot by yourself using the best chatbot builder on the market.
